この図で学生に意識させたいこと
  • 各変数の「役割」：t＝色を確保する代表、q＝価値で勝負する残り。名前と中身が結びつくか。
  • 容器の中身は時間で変わる：同じ q でもステップ 3 と 5 で中身が違う。「今どの時点の q か」を追えるか。
  • 移動＝コード 1 行：箱から箱への矢印が add / poll / addAll に 1:1 対応。
